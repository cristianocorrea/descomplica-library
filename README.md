# Descomplica Library

Exemplo basico de uma api CRUD em .NET C# 

# Dependencias

Instalar dotnet runtime 8

https://dotnet.microsoft.com/en-us/download/dotnet/8.0


# Executar projeto 

Executar via terminal:

- dotnet run --project DescomplicaLibrary.Api/DescomplicaLibrary.Api.csproj

Acessar a url do navegador:

- http://localhost:5056/swagger/index.html
